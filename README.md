# PROGRAMAÇÃO ORIENTADA A OBJETOS

Repositório contendo resoluções de exercicios de atividades, listas e provas, realizadas no decorrer da disciplina de programação orientada a objetos do curso de bacharelado em ciência da computação do IFSP - Câmpus de Presidente Epitácio, realizados por mim, discente Gabriel Pontel de Mori, no primeiro semestre de 2025.
## _Conteúdo Abordado_:

- Revisão de Conceitos: Relembrando aplicações em C agora em Java
- Classes e Objetos:
- Abstração, Generalização e Instanciação,
- Construtores e Sobrecarga de Métodos,
- Encapsulamento,
- Atributos e métodos de classes.
- Herança:
      - Hierarquia de Classes,
      - Overriding.
- Interfaces e Classes Abstratas;
- Classes de Coleções e métodos de interação:
      - Conjuntos, listas e mapas.
- Polimorfismo;
- Tratamento de Exeções;
- Interface Gráfica (Java Swing);
- Acesso a Banco de Dados (JDBC).
